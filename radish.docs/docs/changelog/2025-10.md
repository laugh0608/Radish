## 月度总结

### 核心成果

**第一阶段（已废弃）**：
- ✅ ABP 框架项目初始化
- ✅ OpenIddict 集成与 OIDC 认证
- ✅ Swagger/Scalar API 文档配置
- ✅ Host 首页重构（响应式、主题切换、拖拽排序）
- ✅ Angular 前端移动端适配
- ✅ React 项目 TypeScript 重构

### 技术亮点

1. **OIDC 认证**：OpenIddict 集成，支持多客户端统一认证
2. **API 文档**：Swagger + Scalar 双文档系统，支持多版本
3. **响应式设计**：Host 首页完整的桌面/移动端适配
4. **主题系统**：浅色/深色主题切换与持久化
5. **移动端优化**：底部导航、BottomSheet、鱼眼动效

### 架构决策

1. **ABP 框架**：采用 ABP vNext 作为后端框架（后续废弃）
2. **MongoDB**：使用 MongoDB 作为数据库（后续废弃）
3. **Angular**：前端采用 Angular 框架（后续废弃）
4. **HTTPS 策略**：本地开发统一使用 HTTPS(44342) + HTTP(4200/5173)

### 重要说明

**2025.11.8 项目重建**：
- 完全舍弃 ABP 框架与 MongoDB
- 重新采用 .NET 10 + React + SqlSugar + PostgreSQL 技术栈
- 重新设计项目架构与模块划分
- 本月内容仅作为历史记录保留
