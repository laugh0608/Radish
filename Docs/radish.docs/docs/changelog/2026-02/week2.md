# 2026-02 第二周 (02-10 至 02-16)

## 2026-02-15 (周日)

### 时间显示与用户时区偏好

- **统一时间配置**：后端新增 `Time` 配置段（系统默认时区、展示格式），前后端统一时间展示基线。
- **用户时区偏好持久化**：新增用户时区偏好实体与服务，支持按用户保存时区设置。
- **用户接口补齐**：`UserController` 新增系统时间配置、当前用户时区偏好查询与更新接口。
- **前端接入**：论坛与个人中心时间展示统一使用时区格式化工具，个人中心支持下拉选择并保存时区。
- **跨平台时区兼容**：新增 `TimeZoneResolver`，支持 IANA/Windows 时区 ID 解析与规范化。
- **UTC 存储规范收口**：Repository 层统一规范 `DateTime` 落库为 UTC，避免混合时区写入。

### 萝卜坑（Radish Pit）体验修复

- **字段映射修复**：修复交易记录相关字段映射问题，避免列表展示异常。
- **界面布局重构**：调整萝卜坑页面布局与交易记录区域结构，提升信息密度与阅读体验。
