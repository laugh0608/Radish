# 2026-02 第三周 (02-17 至 02-23)

## 2026-02-23 (周一)

### 评论高亮实时重算修复

- **统一重算入口**：新增 `TriggerHighlightRecheckAsync`，统一神评/沙发实时重算入口。
- **新增评论触发重算**：发表评论后立即触发神评/沙发检查，减少高亮状态滞后。
- **删除评论触发重算**：删除评论后补充触发重算，避免旧“沙发/神评”残留。
- **点赞路径复用**：点赞/取消点赞流程改为复用统一重算入口，并保留后台异步执行。

### 日志链路修复

- **前端日志修复**：修复 `log` 单参数调用时输出 `undefined` 的问题。
- **后端日志修复**：优化 SQL 操作人显示并修复评论点赞奖励日志模板字段。

## 2026-02-22 (周日)

### 论坛搜索交互重构

- **列表页搜索入口改造**：帖子列表改为胶囊式搜索入口，减少与发帖按钮的交互冲突。
- **独立搜索结果页**：新增论坛内搜索视图，支持关键词、排序、时间范围与分页组合检索。
- **时间范围筛选**：后端 `Post/GetList` 增加 `startTime/endTime` 参数，支持按创建时间过滤。
- **搜索摘要可回跳**：搜索摘要条支持点击聚焦到关键词/时间/排序控件，提高二次调整效率。

### 帖子卡片信息增强

- **作者与互动者头像**：帖子列表补充作者头像与最近互动用户头像展示。
- **紧凑神评预览**：卡片展示紧凑神评摘要，提升浏览阶段的信息密度。
- **字段扩展**：新增帖子最新互动者数据契约，支持前端展示“近期互动”信息。

### 商城与通知体验修复

- **商城详情优化**：商品详情首屏调整为左图右信息对齐布局，补足不可购买提示文案兜底。
- **商品类型兼容**：兼容后端返回数字枚举（`1/2/99`）与字符串枚举并统一显示中文类型。
- **通知一致性修复**：修复通知未读状态与评论回复展示不一致问题。
- **界面微调**：Dock 顶部间距微调，提升桌面布局平衡感。

## 本周总结

- **搜索能力升级**：论坛从“列表内搜索”升级为“独立搜索页 + 时间过滤”模式。
- **高亮规则收口**：神评/沙发覆盖新增、删除、点赞三条路径的实时重算。
- **展示体验提升**：帖子卡片、商城详情、通知展示等关键页面获得一致性优化。
