## 差距判定口径

| 类别 | 含义 | 后续处理 |
| --- | --- | --- |
| UI 实现缺口 | 页面路由存在，但布局、密度、主次区域、移动任务流、状态槽、抽屉 / 弹窗或视觉层级未按 Pencil 完整落地 | 进入 D61-D63 成组代码实现 |
| 前端功能缺口 | 设计稿要求的入口、状态、筛选、上下文、动作反馈或只读原因可在现有 API / 路由 / 权限内补齐 | 随对应 UI 批次一起实现 |
| 产品 / API 缺口 | 设计稿包含的能力需要新实时协议、数据模型、权限动作、治理策略或后端 API | 不混入 UI 批次；单独补专题设计 |
| 后置参考 | D38 已裁决后置的公开聊天室、内部调度、内部 Jobs 和独立移动 Console | 不作为当前 D 批次完成条件 |
