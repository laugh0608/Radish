**目标**

### 6. Later：国风 UI / 主题切换 / i18n / 邮件通知系统

**定位**
- 统一归类为影响面更广、复杂度更高的大功能或大治理项
- 暂不作为当前阶段的近期待进入事项

**后置原因**
- 主题切换与国风 UI 会牵动全局视觉和组件体系
- i18n 涉及键值治理、缺失检测与全链路文案收口
- 邮件通知系统涉及模板、触发、投递与运维边界，复杂度较高

**重新启动条件**
- 当前小而快的功能补齐路径完成
- 权限 / 聊天室 / 新功能首版不再频繁返工
- 团队可接受一次较大范围的横向治理改造
