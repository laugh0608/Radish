## 第一开发阶段收口事实

- `2026-04-06`，`v26.3.2-release` 已完成真实发布，首版从“可开发、可联调”推进到“可真实发版、可真实部署、可形成真实回滚预案”
- [首版 dev 边界](/planning/dev-first-scope) 与 [首版 dev 功能矩阵状态表](/planning/dev-first-status-matrix) 当前可视为第一开发阶段范围与结果的归档依据

- `2026-04-07`，官方规划已切换到第二开发阶段：社区深化与多端化

## 第二开发阶段收口事实

- `2026-05-13`，第二开发阶段完成收口评审与归档判断，结论为 Go。
- Web 浏览器方向已收束为公开内容壳层，覆盖 forum / docs / `u/:id` / leaderboard / shop / discover 首批公开路径。
- Android / iOS 方向已收束为 Flutter 移动原生路线，Android MVP 已完成第一轮 RC 验收，`P2-C5-A` 已补齐通知回流与个人复访轻操作首批高价值已登录链路。
- Windows / macOS / Linux 方向已收束为 `Tauri 壳 + WebOS 桌面工作台`，正式签名、自动更新和公开分发事项后置到真实对外分发前。
- 后端 + Console、商城、经验治理、安全尾项、权限种子一致性、构建 warning 与 WebOS 萝卜坑工作流已完成当前收口，后续只处理新暴露的安全 / 授权一致性或主路径 `P0/P1` 问题。
- [第二阶段收口评审](/planning/phase-two-closure-review) 当前可视为第二开发阶段归档依据。
