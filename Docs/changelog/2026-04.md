# 2026 年 4 月开发日志

> 社区体验收口月，重点对齐论坛正文图片、发帖 / 评论 / 聊天室输入体验、通知头像展示与登录后资料刷新体感。

## 本月摘要

- 论坛正文图片坏图链路已收口：`MarkdownRenderer` 当前已统一放行 `attachment://` / `sticker://`，数据库重建后重新发帖也不再因为渲染层过滤导致正文图片损坏。
- 发帖正文、评论区与聊天室输入当前都已接通 `@提及` 联想；评论区与聊天室也已补齐粘贴图片上传，社区输入体验完成一轮统一化收口。
- 编辑帖子默认重新回到正文编辑态，并修复了编辑器只剩单行窄输入区的样式回归。
- 通知中心已为非聊天类通知补齐触发者头像兜底，服务端头像字段缺失时会按公开资料回填，减少默认占位头像。
- OIDC 登录回调完成后会先预热当前用户资料，再返回桌面壳层，缩短头像 / 等级信息迟到刷新的空窗体感。
- 聊天室图片发送成功提示已移除，仅保留失败提示；相关提及弹窗定位也已完成一轮细节收口。

## 当前结论

- 这批社区体验问题已达到“可转维护、可结束本轮主线”的状态。
- 后续若进入下一阶段，应另起议题和计划，不再继续把本轮收口项无限追加到同一波修复中。
