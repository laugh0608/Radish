## 本月摘要

- 聊天室 `P0` 主链路进入可演示状态
- 聊天室 `P1` 已累计补齐引用回复、`@mention`、图片消息、草稿恢复、成员面板、重连补拉与状态条、成员头像来源、乐观发送与失败重试，并完成长整型 ID 精度与输入区布局问题收口
- 身份语义收敛专项从补漏转为统一治理
- 文档 / Wiki 版本历史与回滚落地
- `DbMigrate` 幂等性与种子日志继续收口
- WebOS 桌面应用访问口径已对齐：未登录默认可打开欢迎、组件库、文档、论坛、排行榜、商城等公开应用；聊天室、个人主页、萝卜坑、通知中心、等级继续要求登录；控制台保持独立权限控制
- Wiki 文档三层可见性字段已补齐到 SQLite 自愈链路，`DbMigrate doctor / verify / apply` 现可识别并修复旧库缺少 `Visibility / AllowedRoles / AllowedPermissions` 的情况
- 规划 / 日志入口完成分层瘦身，M12 当前主线切换到权限治理能力
- `Attachment/UploadImage` 共享边界已按方案 B 完成 Sticker 链路最小收口
- 欢迎 App 已新增开源软件说明最小入口，当前采用静态清单方案
- `P3` 论坛投票 MVP 已完成验收收口
- `P4` 论坛问答 MVP 已完成一轮前后端联调收口
- 论坛投票 / 问答列表卡片展示口径已统一，问答采纳异常语义补丁已完成
- `P5` 论坛抽奖 / 浏览记录已完成验收资产补齐与联调问题收口，达到本轮可转维护状态
- `radish.client` 主题与 i18n 首轮主链已从桌面 / 商城 / 论坛扩展到聊天、通知中心与个人中心高频模块
- `radish.client` 文档应用主链、论坛边缘页与深层评论 / 卡片样式已继续收口，欢迎 App 长文案也已完成双语资源化，当前下一步聚焦个人中心残余边缘页与更大范围首版联调
- 国风视觉基线在桌面端深度重构落地：包括衬线大字、云水掩膜、立体画框、水滴动画及基于玉牌隐喻的高密网格应用体系。
- 论坛发帖入口已重构为创作器工作区，评论编辑器与评论弹层也已完成一轮高密度体验收口。
- `@radish/ui` 共享 Toast 已支持自动消失剩余时间进度条，临时提示的时效反馈更明确，且无需逐业务页面重复接线。
- 通知中心已完成包含 Toast 预览、Dock 未读角标、列表、静默已读、删除同步与多端同步的一轮真实 Smoke，并开始从“待联调复核”转向“等待总回归确认”。
- 欢迎 App 已完成平台概览 / 上手路径 / 社区约定 / 开源说明的长文案资源化，语言切换已覆盖欢迎页主体内容。
- 附件业务口径已统一收口到 `attachmentId` / `attachment://{id}` / `/_assets/attachments/*`，换域名时不再需要手工修改附件类数据库数据，相关前后端、功能设计与部署文档已同步更新。
- 论坛发帖创作器已补齐缺失项提示与标签确认引导，欢迎 App 已切到游客安全模式，Console 也已支持站点 favicon 上传、默认图标回退与公开站点设置同步。
- 当前用户缓存 `cached_user_info` 已改为与 access token 身份强绑定的一次性引导缓存，切账号 / 切租户时不再复用上一位用户资料。
- 禁用附件公开访问、个人中心统计聚合、论坛 `hottest` 排序与子评论懒加载过滤当前又完成一轮“治根”治理，社区体验优化开始收敛到真实错态、越权与高负载热点。
