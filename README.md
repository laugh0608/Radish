# Radish

![萝卜](./docs/images/RadishAcg.png)

一个目标远大的应用：后端 .NET 10，前端 React（Vite）。

## 快速开始

- 先决条件
  - .NET 10
  - Node 24+
  - PostgreSQL
- 还原与构建
  - `dotnet restore`
  - `dotnet build Radish.slnx -c Debug`
